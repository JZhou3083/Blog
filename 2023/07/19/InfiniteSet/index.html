离散数学

Introduction

最近重温了一下离散数学，涉及无限集的阅读这里做个总结加强记忆。无限集合这个性质在计算机数学极为重要，属于算法研究的一个重要基础。先简要过一遍常见的几个概念：

  • Domain & Codomain. 主域与从域: The set of all inputs for a function is called the domain; The set of all allowable outputs is called codomain. (输入集是主域，输出集是从域)

  • Surjection, or onto, if each element of codomain is mapped to by at least one element of the domain.

  • Injection, or one-to-one, if each element of the codomian is mapped to by at most one element of the domain.

  • Bijection: one-to-one and onto，一一对应

  • strict: “strictly smaller”

Infinite Cardinality

  1. $A\ strict \ B \iff NOT(A\ surj \ B) \iff \ NOT(|A|\ge|B|) \iff \ |A|<|B|$.
  2. For any sets A, B, C,
    1). $A\ surj \ B \iff B \ inj A$.
    2). $if\ A\ surj\ B\ and\ B\ surj\ C, then\ A\ surj\ C$.
    3). $if\ A\ bij \ B\ and \ B \ bij\ C, then\ A\ bij\ C$.
    4). $A\ bij\ B \iff B\ bij\ A$.
  3. [Schroder-Bernstein] For any sets A,B, if $A\ surj\ B\ and\ B\ surj \ A, then\ A\ bij \ B$.
  4. For all sets A, B, $A\ surj\ B\ or\ B\ surj\ A$.
  5. $A\ strict\ B\ and\ B\ strict\ C$ implies $A\ strict\ C$.
  6. Let A be a set and $b\notin A$. Then $A$ is infinite $\iff A\ bij\ A\cup${b}.

Countable Sets

可数集合也可以说可列集，意思是可以一一列出。可数集有两种，一种是有限集合，一种是可数无限集合。总结起来有几个性质：

  1. A set, $C$ is countably infinite $\iff N\ bij\ C$. set is countable $\iff$ it is finite or countably infinite.
  2. A set, $C$ is countable $\iff N \ surj \ C$. In fact, a nonempty set C is Countable $\iff$ there ia total surjective function $g: N \rightarrow C$.
  3. 常见的countably infinite sets: $Z^+, Z, N\times N,Q^+, Z \times Z, Q$.

Power sets are strictly bigger

Power set, pow(A) is the set of all A’s subsets.Cantor has proved that for any set, $A$, $A\ strict\ pow(A)$.

$pow(N)$ is uncountable. {0,1}$^{\omega}$ is uncountable.

  • If U is uncountable set and $A\ surj\ U$, then $A$ is uncountable.
  • If C is a countable set and $C\ surj\ A$, then $A$ is countable.

Cantor Diagonal Arguments

首先介绍 Finite Binary Strings: $B = $ {$\emptyset, 0, 1, 00, 01, 10, 11, 000, 001, 010,011,…$}.That is, given a finite binary string $b_1b_2b_3…b_k$, its list position is $2^{length}+value(b_1b_2b_3…b_k)$.

所有的计算机程序都是 Finite binary string, 也既是说，$|valid computer programme| \le |B| \le |Infinite|$.

而Cantor 用Cantor Diagonalization反证了Infinite Binary Strings are not countable: $|B_\infty|>|N|$,证明过程可见video.
